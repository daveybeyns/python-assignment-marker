GITHUB UPLOAD STEPS

1. Open the existing python-assignment-marker repository.
2. Choose Add file > Upload files.
3. Upload the replacement root files.
4. Upload the complete assignments folder and examples folder, preserving paths.
5. Confirm that index.html, app.js, styles.css and marker-worker.js are replacements,
   not duplicated files with altered names.
6. Commit to the branch used by GitHub Pages.
7. Wait for GitHub Pages to redeploy.
8. Open:
   https://daveybeyns.github.io/python-assignment-marker/
9. Press Ctrl+F5 to bypass the old browser cache.
